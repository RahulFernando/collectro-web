SMART Garbage Bin Kit : OUR APPROACH TO SOLID WASTE MANAGEMENT

Expandable and Intelligent Waste Management System using Deep Learning and IoT for Modern Organizations

Literature Survey

Waste management is a serious issue in every country, and it directly affects environmental pollution. We discovered modern research studies considering waste classification and proper waste removal empowered with ICT applications.

Waste segregation is an important stage in the waste management systems. Therefore, this has to be done in a more effective and efficient manner. There were several research studies that identification of waste done by using sensors in past. But in modern we could see that so many researchers are using machine learning and deep learning techniques for the automated waste identification. On the other hand, to generate a waste removal path, we need to detect the real-time waste status of the relevant bins, their location, and many other factors. We identified several interesting research studies which explained detection of garbage status and generation of garbage collection routing paths using data collected from sensors attached to bins.

Research Gap

In research studies we could see, CNN for waste segregation, development of specialized data set for the requirement, usage of AI-based cutting-edge devices, collection of real-time status garbage bin status and generation of the optimized collection routing path. But at the same time, we have identified some issues regarding some areas. Real-world scenario of garbage segregation and human error, the possibility of expansion of bin categories based on the future requirement, extended model of waste classification for easy detection and generation of waste collection based on the priority order of bins are some points that researcher haven’t paid much attention. These issues can be defined as our research gap.

Research Problem

After discovery of facts, we have identified some major issues with current manual approach. In Sri Lanka waste segregation happens using human intervention only. However, in that process waste is contaminated due to lack of knowledge, negligence, and misbehavior and so on. We couldn’t see proper guidance for the user to segregate. On the other hand, we could see that lack of punctuality in waste removal. This happens due to having no proper communication real-time waste information. Waste collection staff do not have clear idea regarding the real time waste level status. Using those facts we can conclude our final research problem as , “How to make an intelligent and efficient waste management system which can be customized in any environment?”

Research Objectives

Our main objective is to implement an intelligent and waste management system which can be customized in any environment. We have 4 major sub objectives in our research study as follows ,

  • Creation of intelligent and smart waste segregation classifier system.
  • Enable non-experts to customize the waste management system easily.
  • Measure and predict garbage bin status real time.
  • Generate most optimizing garbage removal path.

Our Solutions

Waste Identification

In our solution we have mounted our web camera on the table, and it bends towards to the floor. User has to point the waste object towards to the camera. Within 5 seconds, it we camera take a picture of the waste object. This is done using a motion detection technique.

Waste Classification

In our solution there are 5 categories. They are food waste , paper , plastic , polythene and other. Waste will be classified under those categories. Undefined (Other) waste will be recorded and further clustered in the cloud environment. Admin can get suggestions from it. Admin can add new waste categories as well.

Smart Bin Kit

In our solution , we have declared a cutting edge device "Smart Garbge Bin Kit". The main core IOT device is the "Jetson Nano Device". Apart from that we have used servo motors , speakers , external power supplies , web camera and dustbins to establish out cutting edge device.

User Guidance

In our solution user has to point the waste object to the web camera. After identifying the waste category , lid of the relavent bin will be opened. At the same time smart garbage bin kit is giving a voice command to put it to the relavent bin. After all , lid will be closed automatically.

Real Time Monitoring

Using ultra sonic sensors and methane gas sensors , we capture real time waste staus. Real time waste staus is displayed in a LCD display of the each bin. At the same time, we can display this information in the dashbaord.

Routing Path

Waste removal routing path will be generated using real time sensor details. We are have defined an function which can generate a most optimized routing path. We use Tree sort and Dijikstra algorithms. We are displaying this waste removal path in a mobile appliction.
